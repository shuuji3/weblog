macOSでEd25519のGPGキーを生成してコミットに署名する

2020-05-05

gpg git github

macOSでGnuPGでEd25519のGPGキーを生成し、Gitのコミットに署名を行います。そして、GitHubに生成した鍵を登録することで、署名が検証されることを確認します。
